# WS2812FX for C

#### 介绍
arduino上的WS2812FX库的C移植版本，与原版略有差异。 1、合并了部分函数名相同参数不同的函数， 2、部分运行参数调整为static类型全局变量 3、需要与 Adafruit NeoPixel for C配合使用[Adafruit NeoPixel for C库](http://gitee.com/landlord_money/adafruit-neopixel-for-c)


#### 软件架构
纯C语言，已在STM32F103上验证

#### 安装教程

1.  下载源码
2.  打开demo文件夹下STM32CubeMX工程文件，按照硬件调整配置
3.  重新生成工程
4.  编译、下载
